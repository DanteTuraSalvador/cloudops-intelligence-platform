## Technologies
- Frontend: Next.js 15, TypeScript, Tailwind CSS
- Backend: Node.js, .NET 8, TypeScript
- Cloud: AWS (Lambda, DynamoDB, API Gateway, S3, CloudFront)
- CI/CD: GitHub Actions, Azure Pipelines
- Infrastructure: AWS CDK/Terraform

## Local Development
```bash
# Install dependencies
pnpm install

# Run tests
pnpm test

# Build all services
pnpm -r build

# Start development
pnpm --filter @cloudops/dashboard dev
```
